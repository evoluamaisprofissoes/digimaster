# ⌨️ DigiMaster — By Evolua+ Profissões

Sistema completo de treinamento de digitação, hospedado no GitHub Pages.

## 🚀 Como usar

Acesse diretamente pelo GitHub Pages:  
**https://seu-usuario.github.io/digimaster**

Ou abra o arquivo `index.html` localmente em qualquer navegador moderno.

---

## 📁 Estrutura do projeto

```
digimaster/
├── index.html          # Página principal
├── css/
│   └── style.css       # Estilos completos
├── js/
│   ├── data.js         # Textos e banco de dados de exercícios
│   ├── keyboard.js     # Teclado virtual interativo
│   ├── stats.js        # Estatísticas e gráficos
│   └── app.js          # Lógica principal
├── assets/
│   └── logo.png        # ⚠️ Substitua pela logo real
└── README.md
```

---

## 🎯 Funcionalidades

- **4 níveis**: Básico, Intermediário, Avançado e Números/Símbolos
- **Métricas em tempo real**: WPM, precisão, erros, palavras digitadas
- **Teclado virtual interativo**: Móvel e redimensionável, indica a próxima tecla
- **Textos sobre tecnologia**: Curiosidades e notícias do mundo digital
- **Estatísticas completas**: Gráfico de evolução, histórico de sessões
- **Persistência local**: Dados salvos no navegador via localStorage

---

## 🖼️ Adicionando sua logo

1. Coloque o arquivo da sua logo na pasta `assets/`
2. Nomeie o arquivo como `logo.png`
3. Recomendado: tamanho mínimo de 200x200px, fundo transparente (PNG)

---

## 🌐 Hospedando no GitHub Pages

1. Faça upload de todos os arquivos em um repositório GitHub
2. Vá em **Settings → Pages**
3. Em "Source", selecione a branch `main` e pasta `/ (root)`
4. Salve e aguarde alguns minutos
5. O site estará disponível em `https://seu-usuario.github.io/nome-do-repositorio`
